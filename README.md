The project is organized into two main tracks reflecting the source repositories:

### 1\. Learning Python

A structured approach to Python programming, covering basics to advanced database integrations.

  * **Basics & Logic**

      * `01_Practice If Else Projects`: Conditional logic exercises (e.g., Age Group Categorization, Ticket Pricing).
      * `02_Practice Loop Projects`: Iterative solutions (e.g., Sum of Even Numbers, Exponential Backoff).

  * **Object-Oriented Programming (OOP)**

      * `05_Object Oriented Programming`: Classes, Inheritance, Encapsulation, Polymorphism, and Static Methods.

  * **Advanced Concepts**

      * `06_Decorators`: Timing execution and debugging function calls.
      * `07_Error_Handling`: Exception handling and file operations.

  * **Database Integration (Projects)**

      * `08_Database_sqlite3`: **YouTube Manager App** using **SQLite** for local data persistence.
      * `10_Python_mongoDB`: **YouTube Manager App** using **MongoDB** (NoSQL).

      * `09_Handling_Api's`: Working with public APIs.
      * `11_Virtual_py`: Managing virtual environments.

## üöÄ Key Projects

### YouTube Manager App

A robust console-based application implemented in two versions to demonstrate different database technologies:

1.  **SQLite Version**: Lightweight, file-based storage for local management.
2.  **MongoDB Version**: Cloud-based storage using `pymongo` to interact with MongoDB Atlas.

**Features:**

  * Create (Add video details)
  * Read (List all videos)
  * Update (Edit video details)
  * Delete (Remove videos)
